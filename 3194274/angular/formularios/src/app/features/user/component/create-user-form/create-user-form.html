Registro

El correo electrónico es obligatorio. Ingresa un correo electrónico válido.
La ficha debe tener mínimo 6 números.
El nombre es obligatorio. El nombre debe tener al menos 2 caracteres.
La contraseña es obligatoria. La contraseña debe tener al menos 8 caracteres. La contraseña debe tener al menos 1 masyúscula, 1 minúscula, un número y un símbolo
La edad es obligatoria. La edad debe ser mayor o igual a 1. La edad no puede ser mayor a 120 años.
