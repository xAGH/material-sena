Registro de Usuario

Completa la información para crear tu cuenta.

El nombre es obligatorio. El nombre debe tener al menos 2 caracteres.
La edad es obligatoria. La edad debe ser mayor o igual a 1. La edad no puede ser mayor a 120.
El correo es obligatorio. Ingresa un correo electrónico válido.
La contraseña es obligatoria. La contraseña debe tener al menos 8 caracteres.
